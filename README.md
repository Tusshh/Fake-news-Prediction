# 📰 Fake News Prediction using Machine Learning

A Machine Learning-based web application that detects whether a news article is **Real** or **Fake** using Natural Language Processing (NLP) techniques. The project preprocesses news text, extracts meaningful features using TF-IDF Vectorization, trains a classification model, and predicts the authenticity of user-provided news articles.

---

## 🚀 Features

- Detects Fake and Real News
- NLP-based text preprocessing
- TF-IDF Vectorization
- Machine Learning Classification
- Interactive Jupyter Notebook implementation
- High prediction accuracy
